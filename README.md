# HWS系外行星異常檢測算法套件

## 🎯 項目簡介
開發三種原創算法，專門用於TESS數據中的系外行星異常現象檢測。

## 🔧 HWS算法家族

### HWS-SRA 飽和修復算法
- **功能**: 解決TESS數據飽和問題，提升數據質量
- **效果**: 數據質量分數從84提升至100
- **應用**: 適用於過亮恆星的數據修復

### HWS-SDA 形狀診斷算法
- **功能**: 區分真實系外行星與食雙星假陽性
- **應用**: 已成功驗證TRAPPIST-1系統凌星形狀
- **特點**: 基於不對稱性分析的形狀特徵提取

### HWS-TAD 時序異常檢測算法  
- **功能**: 發現系外行星系統中的隱藏時序異常
- **發現**: 在Beta Pictoris系統檢測到18個統計顯著異常
- **原理**: 多尺度時序分析結合統計顯著性檢驗

## 📊 科學發現
- **TRAPPIST-1**: 檢測到極顯著時序異常 (p < 0.000001)
- **Beta Pictoris**: 發現18個潛在異常事件，可能與碎屑盤活動相關
- **方法驗證**: 在多個已知系統中驗證算法有效性

## 🚀 技術特點
- 專門針對TESS數據特性優化
- 完全自主研發的算法框架
- 經過真實天文數據驗證
