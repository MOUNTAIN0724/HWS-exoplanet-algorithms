# 算法技術規格

## HWS-SRA 飽和修復算法
**技術核心**: 創新的數據重建技術
**適用場景**: TESS過亮恆星的飽和問題
**輸入輸出**: 
  - 輸入: 飽和光變曲線
  - 輸出: 修復後的光變曲線
**關鍵參數**: 飽和閾值、重建窗口大小

## HWS-SDA 形狀診斷算法  
**技術核心**: 凌星輪廓分析的形狀特徵提取
**特徵輸出**: 不對稱度、傾斜率、曲率指標
**應用價值**: 識別非標準凌星形狀

## HWS-TAD 時序異常檢測算法
**技術核心**: 多尺度時序信號處理 + 統計顯著性檢驗
**檢測能力**: 軌道調製、時序異常、周期變化
